# Avantia Health Optimization Website

Board-certified health coaching website built with Next.js 14, Tailwind CSS, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

This site is configured for deployment on Vercel. Connect the repository to Vercel for automatic deployments.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons
